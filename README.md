# blog_application

A simple and functional Blog Web Application built using Django.
Users can register, log in, create posts, and manage their own content.

## Features
- User Authentication (Login & Signup)
- Home page displaying all blog posts
- Create new blog posts
- View personal posts (My Posts)
- Logout functionality
- Automatic post timestamp
- Author-based post management

## Tech Stack
- Backend: Django (Python)
- Database: SQLite (default Django DB)
- Frontend: HTML, CSS (Django Templates)
- Authentication: Django built-in auth system
